# Edna Moda Appointment Management System

## Project Overview
This project is an individual academic software development project for **1║ DAW ¢ Proyecto Integrador 2025¢2026**.

The application is designed for **Edna Moda's fashion workshop** to manage tailoring appointments, clients, suits, and workshop rooms. It provides a desktop-based management system where Edna can log in and perform the main administrative operations needed in her studio.

## Main Features
- Secure login for Edna
- Manage clients
- Manage appointments
- Manage workshops
- Manage suits related to each client
- View and organize scheduled appointments
- Apply business rules for hero/villain scheduling (bonus)

## Project Goals
The goal of this project is to develop a Java application following good software engineering practices, including:
- Requirement analysis
- Relational database design
- MVC architecture
- UML modeling
- Version control with GitHub
- Agile planning with Trello
- Technical documentation

## Technologies
- **Java**

## Proposed Architecture
The application will be organized using the **MVC pattern**:
- **Model**: application data and business entities
- **View**: graphical user interface
- **Controller**: application logic and interaction flow

## Core Entities
- Client
- Suit
- Appointment
- Workshop
- User/Login

## Functional Scope
The system allows Edna to:
- Create, update, delete, and view clients
- Create, update, delete, and view appointments
- Create, update, delete, and view workshops
- Associate appointments with clients, suits, and workshop rooms
- Prevent consecutive appointments between heroes and villains without a one-hour gap

## Development Methodology
This project follows an **agile approach inspired by Scrum**, adapted to an individual workflow. Development is divided into sprints, with each sprint focusing on analysis, design, implementation, testing, and documentation.
